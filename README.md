# 🍔 Delisas Food - Premium Delivery App

Delisas is a high-performance, aesthetically pleasing food delivery application built with **Flutter**. It features a state-of-the-art **Glassmorphic UI**, real-time simulated order tracking, and a seamless user experience designed to wow users at first glance.

---

## ✨ Key Features

- **🚀 Modern UI/UX**: Built with a custom design system featuring glassmorphism, smooth gradients, and premium micro-animations.
- **🛰️ Real-time Tracking**: Interactive delivery tracking screen with animated couriers, route paths, and live status updates.
- **🔐 Session Persistence**: Secure login system with `shared_preferences` that keeps you logged in even after refreshing.
- **🛍️ Cart & Favorites**: Full functionality to manage your favorite meals and track your orders.
- **🖼️ Smart Asset Loading**: Automatically handles both local assets and high-resolution network images with fallbacks.
- **🌓 Dynamic Transitions**: Custom fade and scale transitions between all screens for a native app feel.

---

## 🛠️ Technology Stack

- **Framework**: Flutter (Web, Android, iOS)
- **State Management**: Provider
- **Storage**: SharedPreferences
- **Theming**: Custom Vanilla CSS-style Theming
- **Icons**: Material Icons & Cupertino Icons
- **Fonts**: Google Fonts (Inter, Outfit)

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (latest stable version)
- A code editor (VS Code, Android Studio)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/delisas-food-app.git
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run -d chrome
   ```
